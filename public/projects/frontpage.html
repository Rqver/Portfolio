Frontpage

Aggregates the front page of NZ Media outlets into a single view.

Live Site Source Code

About Frontpage

Frontpage gives you the ability to view the real-time top stories of NZ media outlets in one place.

The eight included outlets (Stuff, RNZ, NZ Herald, The Post, The Spinoff, Newsroom, NBR & Business Desk) are frequently polled to return the top stories on their respective home pages, in order of importance/how they appear on their home page.

Visitors of Frontpage can customize what outlets appear, and in what order they appear on the page. The site is fully responsive, and designed to be extremely self-explanatory and easy to use.

Why

I have an interest in the news media, and frequently found myself visiting eight different websites multiple times an hour.

A lot of these outlets do offer RSS feeds, however these RSS feeds sort content by latest published, whereas I wanted to replicate the experience of visiting each of these outlet's home page, and view the stories in the order each outlet decided to present them at any given time.

Frontpage is an interesting website to watch amidst significant breaking news, as to what outlet is first to a story, how each outlet reports on that story, etc. etc.

Tech Stack

  • Written in TypeScript using Deno as a runtime.
  • The frontend is built using Tailwind CSS.
