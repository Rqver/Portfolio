Arataki

A deployment manager designed for use over MCP.

About Arataki

Arataki is a self-hosted deployment & process manager that runs all of my personal projects, including the website you are reading this on. It replaced a manual loop of SSHing into a server, pulling, restarting, and hand-writing nginx & certbot configs with a single web panel and an MCP server.

Deploying a new app is done by picking a GitHub repo and optionally a subdomain; Arataki allocates a port, writes the systemd unit, creates the DNS record in Cloudflare, issues a TLS certificate, and sets up nginx. Services can be public websites, internal APIs, or headless workers, and each service has a page with live status, CPU & memory use, logs, environment variables, and a full deploy history.

Arataki also manages each app's SQLite databases. Managed databases are continuously replicated to object storage by Litestream, and can be browsed and queried (read-only) directly in the panel.

Everything the panel can do is also exposed as tools over MCP. Claude can deploy a project, restart it, read logs from a live project, query a projects live database, check a projects resource usage, etc.: "Commit, push and redeploy via Arataki" or "There's an issue with the database on X app, explore it via Arataki"

MCP actions are recorded in the same audit trail as panel actions, environment variable values are never exposed, and each database has to be explicitly opted-in to being queryable over MCP.

Why

The number of projects I self-host kept growing (seventeen services at the time of writing), and every new project meant SSHing into the server to hand-write an nginx config and a systemd unit, and set up DNS records & TLS certificates. Every update to an existing project meant SSHing in to pull and restart it.

The bigger motivation though was MCP. I manage/do a lot through Claude, and I wanted my server to be something an AI agent could safely operate.

Tech Stack

  • Written in TypeScript using Deno as a runtime.
  • The frontend is built using Tailwind CSS.
  • SQLite is used as a database.
  • The MCP server is built using the official MCP TypeScript SDK.
  • On the server, Arataki drives systemd, nginx, certbot, Cloudflare DNS, and Litestream backups to Cloudflare R2.
