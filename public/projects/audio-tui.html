Audio TUI

A TUI to manage playback of radio streams and Plexamp playlists.

Source Code

About Audio TUI

Audio TUI gives you the ability to, using only a keyboard, stream a radio station or Plexamp playlist.

The project is designed for Linux desktops and compatibility with Alexays/Waybar. The TUI is aware of what is currently playing on each of your radio stations, and knows of all your Plexamp playlists. It supports vim motions (j/k) and the arrow keys.

Why

This project was primarily built for myself, and my want for the ability to manage my music/radio faster. On my Arch desktop, I have Audio TUI binded to Super + M, and can access it anywhere.

This is a very niche project, but I did opt to open source it/provide instructions for using it locally in the event someone else was ever in the same situation.

Tech Stack

  • Written in TypeScript using Deno as a runtime. Compiled to an executable for runtime.
